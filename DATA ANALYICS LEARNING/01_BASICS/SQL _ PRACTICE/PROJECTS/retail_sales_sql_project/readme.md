# Retail Sales Analysis using SQL

## Objective
Analyze retail sales data to identify revenue patterns, high-performing products,
and customer spending behavior using SQL.

## Tools Used
- MySQL
- SQL (Joins, Aggregations, GROUP BY, HAVING)

## Project Structure
- schema.sql – database design
- data.sql – sample transactional data
- analysis.sql – business analysis queries
- insights.md – insights derived from analysis

## Key Outcomes
- Identified high-revenue regions
- Analyzed product performance
- Discovered high-value customers
